# Introduction to Low Level Design

## What is LLD?
- Designing classes, objects, relationships between them
- Focus: HOW the code is structured, not just WHAT it does

## LLD vs HLD
- LLD = class diagrams, methods, OOP
- HLD = servers, databases, APIs, scale

## Key Questions to Ask in an Interview
- What are the entities?
- What state does each entity hold?
- What behavior does each entity have?
